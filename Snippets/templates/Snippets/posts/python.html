Insight to Python

Python, one of the easiest language to implement in today's time. Python comes with various built in libraries and external libraries as well. To name one of them , Numpy is a library which deals with array based problems. Matplotlib is an another library, one of my personal favorite, is meant to deal with data visualization.


Enough with the introduction as I am aware that not all of you would be coders. However, let me tell you that a programming language is just mathematics and logic written with a particular syntax.


Now I know some of you may not be interested in logic either but python is not just used for programming sake and applying machine learning algorithms but it can be used for even art purpose. Recently, I made an audio visualizer just with a few lines of code with Numpy and Matplotlib.


"Python is not just what you think it is, it is so great that nowadays most of the people believe in using python for automating their boring stuff in their laptops and computers."

From this article, I am just hoping that the reader would open their laptop and just install python and a few other required libraries.You are free to ask me any questions dealing with its installation or anything related to python.
